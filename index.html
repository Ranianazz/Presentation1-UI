Kids' Magic Song Maker

Interactive UI responding to the “No Text Characters” prompt

Welcome

Created by Rania Nazzal
An interactive prototype blending shapes, colour and sound.

Conceptual Response

This project responds to the “No text characters” prompt. The interface communicates only through shapes, colours, and sounds.

  • Inspired by playful children’s toys and music boxes
  • Designed for ages 2–5 who cannot yet read
  • Focuses on fun, creativity, and sensory engagement

Technical Approach

Built with p5.js and p5.sound.

  • Click/tap → generates shape + note
  • Notes follow a pentatonic scale & “Twinkle Twinkle” sequence
  • Shapes expand & fade for visual feedback
  • Randomness (Math.random()) adds variation in size, colour & duration

System Map:

Peer Testing Feedback

Prompt 1: Intuitive Mapping

Peers found clicking to make sound intuitive: “Simple, fun, and effective!”

Prompt 2: Cohesion

Playful visuals + melody worked well, but some colours clashed. Suggestions included smoother fade-outs & clearer pitch mapping.

Response & Next Steps

  • Refine colour palette for consistency
  • Add more shapes and smoother dissolves
  • Map click position to pitch (left = low, right = high)
  • Introduce simple user choice (toggle melodies/shapes)
